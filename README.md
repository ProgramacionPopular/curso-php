# Curso de PHP

Hola! esta es la página de bienvenida al curso de PHP,
para seguir el curso simplemente hay que abrir la línea de comandos
y moverse hasta esta carpeta con el comando "cd" y luego escribir:

```
php -S 0.0.0.0:8000
```
